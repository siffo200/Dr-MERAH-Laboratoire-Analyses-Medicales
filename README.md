# Dr. Merah - Medical Laboratory Appointment System 🏥

## Overview
A streamlined web application developed for **Dr. Merah Medical Analysis Laboratory**. This system automates the appointment booking process, directly linking the patient-facing website to a centralized Google Sheets dashboard for the reception desk.

## Key Features 🚀
*   **Patient Booking Interface:** A user-friendly web interface for patients to request medical analysis appointments.
*   **Automated Data Sync:** Seamlessly integrates with Google Sheets API to log all appointment details in real-time.
*   **Reception Dashboard:** Receptionists receive instant updates on a structured spreadsheet, eliminating manual data entry and reducing booking conflicts.
*   **Efficient Workflow:** Simplifies the communication between patients and the laboratory staff.

## Tech Stack 💻
*   **Frontend:** HTML, CSS, JavaScript.
*   **Backend & Integration:** Google Sheets API.
